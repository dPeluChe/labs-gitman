# Keyboard Shortcuts

GitMonitor provides comprehensive keyboard shortcuts for power users.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘K` | Focus search bar in sidebar |
| `⌘R` | Refresh all projects |
| `⌘N` | Add new monitored path |
| `⌘,` | Open Settings |
| `ESC` | Close sheets/modals (Add Path, Settings, etc.) |

## Project View Shortcuts

When viewing a project:

| Shortcut | Action |
|----------|--------|
| `⌘1` | Switch to Overview tab |
| `⌘2` | Switch to Files tab |
| `⌘3` | Switch to Terminal tab |
| `⌘4` | Switch to Branches & History tab |
| `⌘R` | Refresh current project |
| `↑` | Navigate to previous project |
| `↓` | Navigate to next project |
| `ESC` | Return to dashboard |

## Menu Bar Shortcuts

Click the menu bar icon to:
- Quick view all projects
- See project counts (total, with changes, PRs)
- Refresh all projects
- Open main window
- Quit app

## Tooltips

All buttons in the UI now have helpful tooltips:
- **Add Path** button shows "⌘N"
- **Scan** button shows "⌘R"
- **Refresh** button in project view shows "⌘R"
- Action buttons show their titles on hover

## Tips

1. **Fast Navigation**: Use `↑↓` arrows to quickly browse through projects
2. **Tab Switching**: `⌘1-4` lets you jump between detail tabs instantly
3. **Search**: `⌘K` is the fastest way to find a specific project
4. **Refresh**: `⌘R` works globally (refreshes all) or per-project (refreshes current)
5. **Escape**: `ESC` closes any modal or returns to dashboard from project view
6. **Branches & History**: The Branches tab now includes a toggle to switch between branch list and commit timeline
